- **.gitignore**: Git 버전 관리에서 제외할 파일 및 폴더 목록을 정의합니다. `node_modules`나 `dist`와 같이 저장소에 올릴 필요가 없는 파일들을 지정합니다.
- **manifest.json**: Chrome 확장 프로그램의 핵심 설정 파일입니다. 확장 프로그램의 이름, 버전, 권한, 아이콘, 실행 스크립트(background, content) 등을 정의합니다.
- **package.json**: Node.js 프로젝트의 메타데이터와 의존성 목록을 관리하는 파일입니다. 프로젝트 이름, 버전, 스크립트(예: 빌드, 테스트) 및 `dependencies`, `devDependencies`를 포함합니다.
- **package-lock.json**: `package.json`을 기반으로 설치된 `node_modules`의 정확한 트리 구조를 기록합니다. 이를 통해 다른 환경에서도 동일한 버전의 패키지를 설치할 수 있습니다.
- **webpack.config.js**: JavaScript 모듈 번들러인 Webpack의 설정 파일입니다. 여러 개의 JavaScript 파일을 하나로 합치거나, 코드를 압축하고, 개발 서버를 설정하는 등의 작업을 정의합니다.
- **dist/**: Webpack 빌드 프로세스를 통해 생성된 최종 결과물(번들된 파일)이 저장되는 폴더입니다. 실제 확장 프로그램이 사용하는 파일들이 이곳에 위치합니다.
- **fonts/**: 확장 프로그램에서 사용하는 폰트 파일(예: `PeachMarket-Regular.ttf`)을 보관하는 폴더입니다.
- **node_modules/**: `package.json`에 명시된 모든 npm 패키지들이 설치되는 폴더입니다.
- **src/**: 프로젝트의 주요 소스 코드가 위치하는 폴더입니다.
- **src/background.js**: 확장 프로그램의 백그라운드에서 실행되는 스크ript입니다. 이벤트 리스닝, 상태 관리 등 보이지 않는 곳에서 동작하는 로직을 처리합니다.
- **src/content.js**: 웹 페이지의 컨텍스트 내에서 직접 실행되는 스크립트입니다. DOM 조작, 페이지 내용 추출 등 웹 페이지와 직접 상호작용하는 기능을 담당합니다.
- **src/toggleToolbar.js**: 툴바 UI를 표시하거나 숨기는 기능을 담당하는 스크립트로 추정됩니다.
- **src/modules/**: 기능별로 분리된 JavaScript 모듈들이 위치하는 폴더입니다.
- **src/modules/changeTextStyle.js**: 텍스트 스타일(글꼴, 크기, 색상 등)을 변경하는 기능을 담당하는 모듈입니다.
- **src/modules/extractors.js**: 웹 페이지에서 특정 정보(예: 본문 내용)를 추출하는 로직을 담고 있는 모듈입니다.
- **src/modules/readerMode.js**: '읽기 모드' 기능을 구현하는 모듈입니다. 광고나 불필요한 요소를 제거하고 본문만 깔끔하게 보여주는 역할을 합니다.
- **src/modules/readingGuide.js**: 사용자의 읽기를 돕는 가이드(예: 하이라이트 라인) 기능을 담당하는 모듈입니다.
- **src/styles/**: 스타일 관련 파일(CSS, HTML)을 보관하는 폴더입니다.
- **src/styles/toolbarCSS.js**: 툴바의 CSS 스타일을 JavaScript 문자열 형태로 가지고 있는 파일입니다. 동적으로 스타일에 주입하기 위해 사용됩니다.
- **src/styles/toolbarHTML.js**: 툴바의 HTML 구조를 JavaScript 문자열 형태로 가지고 있는 파일입니다. 동적으로 페이지에 삽입하기 위해 사용됩니다.
